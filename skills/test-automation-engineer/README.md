# test-automation-engineer

Test automation engineering workflow for turning high-value checks into stable, maintainable automated tests at the right level of the stack.

See [SKILL.md](./SKILL.md) for the full workflow.

## What it does

- Chooses between unit, integration, API, contract, UI/e2e, smoke, and other checks.
- Applies test pyramid thinking and identifies when not to automate.
- Designs maintainable fixtures, selectors, assertions, CI integration, and reporting.
- Collaborates with [`software-engineer`](../software-engineer/), [`manual-tester`](../manual-tester/), and [`product-owner`](../product-owner/).
